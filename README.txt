자기소개서

Oracle/GIS 데이터 전문가, TMap POI 데이터 구축 및 운영 담당

저는 SKT TMap 서비스의 POI 데이터 구축 및 운영을 담당한 Oracle/GIS 데이터 전문가입니다. 
SK그룹의 텔레매틱스 사업부가 태동하던 시기부터 개발자로 합류하여, 
23년 동안 배치 서버, Oracle DBMS, 델파이, Java Spring 등 다양한 기술을 다루며 서비스의 안정성을 유지해 왔습니다.
SK 담당 회사와 책임자가 바뀌는 동안에도, 
저는 변함없이 서비스의 연속성을 지키고 운영을 최적화하는 역할을 수행해 왔습니다. 
맡은 업무에 대한 주인의식이 시스템을 더 발전시키고 견고하게 만들었다고 자부합니다.
앞으로도 저는 주어진 역할에 그치지 않고, 
축적된 경험과 기술을 바탕으로 서비스의 지속적인 발전과 최적화를 이끌어 나가는 개발자로 성장하고자 합니다. 
변화하는 환경 속에서도 흔들림 없이 맡은 바를 수행하는 저의 경험과 역량이, 
새로운 도전 속에서도 큰 가치를 발휘할 것이라 확신합니다.



주요 경험 및 역량:
- POI/주소 데이터 수집, 가공, 배포 시스템 설계 및 운영
 . (pro*c, spring framework mybatis)

- 대용량 Oracle DB 설계/관리 및 SQL 튜닝

- 실사툴, 편집툴 등 GIS 데이터 운영 시스템 개발 
  . 무인단속카메라/POI 위치 수집 실사툴 
  . POI 데이터 편집/운영툴 
  . 안전운전도우미 데이터 운영툴 
  . 유가정보-도로네트웤 지도 매핑툴 

기술 스택:
- 언어: Delphi, Java(Spring Framework, Spring Boot), Unix-C, Pro*C
- DB: Oracle
- 기타: QGIS, ArcView, MapObects

특화 역량:
- 대용량 공간정보 데이터 처리 및 최적화
- 위치기반 서비스 운영 시스템 설계/개발
- Oracle DB 테이블 설계 및 SQL 최적화와 데이터 정합성 관리.

1. 소개 및 개요
안녕하세요, 저는 모바일 길안내 서비스인 TMap시스템의 초기부터 최근까지 20년 이상 일해 온 경험을 보유한 IT 개발자입니다. Linux 서버, Pro*C, Java(Spring Framework)를 활용하여 Oracle Database의 대용량 데이터를 처리하고, 테이블 설계 및 데이터 정합성 처리에 전문성을 가지고 있습니다. 이 포트폴리오는 제가 참여했던 주요 프로젝트와 기술적 기여를 소개합니다.
